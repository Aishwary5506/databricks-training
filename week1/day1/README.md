## 🟢 Week 1 - Day 1

### 📌 Topics Covered

* Basic SQL Queries (`SELECT`)
* Filtering using `WHERE`
* Pattern matching using `LIKE`
* Date-based queries
* Aggregate functions (`SUM`, `AVG`, `MIN`)
* Grouping using `GROUP BY`
* Filtering groups using `HAVING`
